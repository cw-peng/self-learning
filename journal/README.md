# Journal — Daily Learning Log

Write one entry per learning day. This is your learning record, not a time tracker.

## Why Journal?

- **Memory:** Writing solidifies learning. You'll remember more.
- **Patterns:** Over time, you'll see what works and what doesn't.
- **Feedback:** `/planner` reads your journal to give better recommendations.

## How to Create an Entry

Use `/journal` at the end of each learning session. It will:
1. Ask reflective questions about your session
2. Generate a structured `.md` file using the template
3. Save it as `YYYY-MM-DD.md`

Alternatively, copy `.claude/templates/journal-entry.md` and fill it in manually.

## What Goes In a Good Entry

- **Today's summary** — what you actually did (not what you planned to do)
- **Key insights** — the "aha!" moments
- **Questions and difficulties** — be specific about what confused you
- **Knowledge gaps** — what you realized you don't know
- **Tomorrow's plan** — one concrete next step

## Review Habit

Every {{REVIEW_DAY}}, skim the week's entries. Patterns will emerge.
