# Review — Progress & Quality Records

Regular review turns experience into insight.

## Review Types

### Daily (via `/journal`)
- What did I learn?
- What was difficult?
- What's next?

### Weekly (every {{REVIEW_DAY}})
- Progress vs. roadmap
- Knowledge gaps
- Adjust plan if needed

Use the template in `.claude/templates/review-template.md`. Save as `week-N.md`.

### Per-Project
- After completing a project in `{{DIR_PROJECTS}}/`
- What went well? What was harder than expected?
- Update the project's README with learnings.

### Per-Milestone
- After completing a roadmap phase
- Run `/quiz` to assess retention
- Run `/planner "Milestone review"` to plan the next phase

## Review Template

Copy `.claude/templates/review-template.md` and fill it in, or use `/planner "Weekly review"`.
