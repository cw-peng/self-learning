# Resources — Curated References

Books, tutorials, documentation, videos, and other learning materials for {{LANGUAGE}}.

## Why Curate?

The internet is infinite. Without curation, you'll drown in tabs. This directory is your personal, vetted reference list.

## Organization

Organize however makes sense for you:

```
resources/
├── books.md          (books with notes on which chapters matter)
├── tutorials.md      (online tutorials and courses, ranked by quality)
├── documentation.md  (official docs, API references, cheat sheets)
├── videos.md         (talks, lectures, channels)
├── tools.md          (useful tools, linters, debuggers for {{LANGUAGE}})
└── papers.md         (academic papers, if relevant)
```

## What Makes a Good Resource Entry

```markdown
### [Title](URL)
- **Type:** Book / Tutorial / Video / Docs / Tool
- **Quality:** ⭐⭐⭐⭐⭐ (your rating after actually using it)
- **Why it's good:** [One sentence — what makes this worth your time]
- **Best for:** [Beginner / Intermediate / Advanced — be honest]
- **Time investment:** [Hours to complete, or reference-only]
- **Notes:** [Specific chapters, caveats, what to skip]
```

## Ground Rules

- **Rate only what you've actually used.** Don't collect; curate.
- **One entry per resource.** No duplicates.
- **Delete outdated resources.** If the tutorial is for an old version, remove it.
