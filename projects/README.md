# Projects — Applied Learning

Build real things. Projects are where isolated skills become integrated understanding.

## Project vs. Exercise

| Exercise | Project |
|----------|---------|
| Focuses on one concept | Combines multiple concepts |
| Takes 30-60 minutes | Takes days to weeks |
| Has a known solution | Has many possible solutions |
| Practice | Application |

## Starting a Project

1. Run `/planner "Project: [idea]"` to scope it appropriately.
2. Copy `template/` as a starting structure.
3. Define clear success criteria before starting.
4. After completion: `/reviewer` on the project, then `/journal` to capture lessons.

## Project Ideas Template

When brainstorming projects, aim for something that:
- Uses 2-3 concepts from your roadmap
- Has a clear "done" state (avoid open-ended projects early on)
- Is interesting to YOU (motivation matters more than portfolio value)
- Can be completed in days, not months
